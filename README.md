# PBay
A show &amp; movie service using torrents from ThePirateBay

#### API for fetching the torrents:
https://rapidapi.com/volodimir.kudriachenko/api/ThePirateBay

ThePirateBay.search gets the torrent id
ThePiratebay.getTorrent gets the magnet link from the torrent id

#### API for playing torrents:
https://webtorrent.io/docs
